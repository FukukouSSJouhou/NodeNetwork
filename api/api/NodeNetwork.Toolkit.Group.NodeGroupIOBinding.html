Class NodeGroupIOBinding

Facilitates connections between nodes outside and inside a group. This is performed by having inputs on the group node (in the supernet) that map to outputs on (mostly) the EntranceNode in the subnet. Likewise, outputs of the group node map to inputs on (mostly) the ExitNode in the subnet.
